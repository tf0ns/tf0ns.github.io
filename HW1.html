HW 1

In this assignment students create a basic webpage using HTML and CSS. Starting with the homepage titled CNIT 131a XML and JSON Assignments ,you will then create a link for two sepperate homework assignments, this can be as a table or as a list.

For the table follow these instructions

  1. Have two columns, the first column will contain name of the homework assignment
  2. In the second column put a summary of the assignment or a comment about how it went for you doing the homework.

For the list follow these instructions

  1. Make two hyperlinks, one per homework assignment.
  2. For each one write a description of the assignment or a comment about your experience with the assignment.
